# Tuition Payment Reminder System

A Spring Boot application designed to manage student tuition payments and send reminders.

## Features
- Add and manage student details
- Track tuition payment status
- Admin and student views
- MySQL database integration
- Email reminder service

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- Maven

## Project Structure
- Controller layer for handling requests
- Service layer for business logic
- Repository layer for database access
- Thymeleaf templates for UI

## How to Run
```bash
mvn spring-boot:run
